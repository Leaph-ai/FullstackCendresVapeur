# Frontend

Interface web construite avec **React 19**, **TypeScript** et **Vite**.

## Prérequis

- [Bun](https://bun.sh/) (recommandé) ou Node.js 18+

## Installation et démarrage

### Avec Bun (recommandé)

```bash
# Installer les dépendances
bun install

# Démarrer le serveur de développement
bun dev
```

### Avec npm

```bash
# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev
```

## Accès

Une fois démarré, l'application est disponible sur : http://localhost:5173

## Scripts disponibles

| Commande           | Description                        |
|--------------------|------------------------------------|
| `bun dev`          | Serveur de développement avec HMR  |
| `bun run build`    | Build de production                |
| `bun run preview`  | Prévisualiser le build de production |
| `bun run lint`     | Linter le code avec ESLint         |
